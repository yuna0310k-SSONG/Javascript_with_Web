## 🐹 두두팡 (DoDoPang)

랜덤으로 튀어나오는 두더지를 클릭해 점수를 얻는 미니게임 🎯  
황금 두더지는 점수 ×2 💛, 폭탄은 점수 0점 💣

### 🎮 게임 방법

- 난이도(쉬움/보통/어려움) 선택
- 시간(초) 입력 후 **게임 시작** 클릭
- 일반 두더지: +1점 ｜ 황금 두더지: 점수 ×2 ｜ 폭탄: 점수 초기화

### ⚙️ 개발 포인트

- HTML · CSS · JavaScript로 제작된 브라우저 게임
- 난이도/시간 조절 지원, 반응형 UI
- `setTimeout`/`setInterval` 기반 랜덤 스폰 로직
- CSS 애니메이션(황금 반짝/폭탄 피드백) 적용
